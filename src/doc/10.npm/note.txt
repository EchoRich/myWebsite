 ##  如何定义全局包（包是模块的集合 ）
  在命令行中使用的 
  包必须要有package.json
  npm init  -y  这个里面就会有一个package.json了 
找到对应的文件夹  然后npm link 
#! /usr/bin/env node 这个是代表在当前的node环境中执行 


 ## 发布包   
 先去官网上 nrm (node 的源管理器) npm nvm  版本管理器 
 
npm link  就是链接本地包 
 npm install nrm  -g

 nrm  ls // 查看所有的仓库源
 nrm use  npm    // 可以更改使用的s仓库 后面可以通过npm config list 进行查看

 nrm  test   会对所有的包进行测试
